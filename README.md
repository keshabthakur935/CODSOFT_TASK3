# Dynamic Calculator

This project is a **dynamic calculator web application** built using **HTML, CSS, and JavaScript**. The calculator performs basic arithmetic operations and provides a simple, interactive user interface for performing calculations quickly.

## About the Project

The Dynamic Calculator is designed to simulate a real calculator on a webpage. It allows users to perform different mathematical operations with a clean layout and responsive design. The project demonstrates the use of **JavaScript for handling user input and performing calculations dynamically**.

## Features

* Perform basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
* Interactive buttons for numbers and operators
* Real-time display of input and results
* Backspace and clear functionality
* Responsive design for mobile and desktop
* Smooth and user-friendly interface

## Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and layout design
* **JavaScript** – Logic for performing calculations and handling user interactions

## Functionalities

* Number input
* Operator selection
* Clear (C) button
* Backspace/Delete function
* Decimal number support
* Real-time calculation display
* Scientific calculator
